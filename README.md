**概要**  
物理的に存在する物品から図面や文書等のデータまで、幅広い「モノの管理」に対応する在庫/データ管理システムです。  
検索機能/タグ・コメント追加
/在庫出し入れ状況/在庫数管理/バックアップの作業を統合的に行うことを目的に開発中です。
